# SecureWave - Cybersecurity Landing Page

SecureWave is a modern and responsive cybersecurity landing page built to strengthen my frontend development skills using HTML and CSS.

This project was created as practice to reinforce fundamental concepts while applying more advanced HTML structure and styling techniques.

## Features

- Semantic HTML5 structure
- Responsive design
- Modern UI layout
- CSS Flexbox and Grid
- Interactive hover effects
- Pricing section
- Contact form
- FAQ section
- Smooth visual hierarchy

## Technologies Used

- HTML5
- CSS3

## Purpose

The main goal of this project is to practice and improve:

- Advanced HTML structuring
- CSS layout systems
- Responsive web design
- Clean code organization
- Modern landing page design principles
